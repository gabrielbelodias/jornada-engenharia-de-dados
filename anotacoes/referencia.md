# O que é Git
- Git é repositório para salvar os arquivos e manter a rastreabilidade do código
# O que é commit
- Commit é tirar um print da história em ponto específico.
# O que é branch
- Segue principio de desenvolver em paralelo novas funcionalidades ou fazer teste sem alterar a main
# Por que nunca subir senha
- Questão segurança, caso quebre ou tenha um ataque as pessoas consegue essas info.
